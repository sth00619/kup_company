종료 시간 설명
X 영상이 0초부터 시작하여 끝까지 재생됩니다.
O 영상이 0초부터 시작하여 종료 시간까지 재생됩니다.
21:34 영상이 0초부터 시작하여 21분 34초간 재생됩니다.
입력한 종료 시간이 영상의 길이보다 길 경우 종료 시간은 작동하지 않습니다.
